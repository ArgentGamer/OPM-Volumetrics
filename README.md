# OPM-Volumetrics
Volumetric Clouds for Outer Planets Mod in Kerbal Space Program.

The source does not include the textures because of the file size limit, but the releases will have them. Please download the releases.

Dependencies are Stock Volumetric Clouds V5 and it's dependencies.
Do not install any other mods that add Scatterer and EVE configs to Outer Planets mod such as Pood's OPM-VO, otherwise there will be a conflict.
